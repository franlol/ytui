# OpenCode Runtime Definitions

This directory contains the OpenCode runtime layer of the repository's executable agentic system definitions.

- `agents/`: custom agent definitions (`*.md` with frontmatter)
- `commands/`: slash commands (`*.md` with frontmatter and prompt template)
- `skills/`: skill folders with `SKILL.md` and required frontmatter

These files complement policy/spec docs in `docs/`.
Claude Code CLI adapter definitions live in the parallel `.claude/` runtime layer.
