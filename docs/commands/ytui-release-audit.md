# /ytui-release-audit

## Purpose

Final gate before merge/release.

## Skills

- `ytui-release-auditor`
- `ytui-architecture-guardian`
- `ytui-plugin-reviewer` (if plugins changed)

## Required Checks

- mandatory tests pass
- config compatibility maintained
- docs updated
- no blocking policy violations

## Required Output

- audit checklist results
- open risks
- `PASS/WARN/FAIL`
